The Phonetic Pashto keyboard for Mac maps Pashto characters to English keyboard keys based on their phonetic similarity, making it intuitive for users who are familiar with both languages.

Follow the following steps to download, install, and use it on a Mac.

## Installation Process

### Step 1: Download Installer

1. Click on the <a href="/PashtoDAOPKeyboardInstaller.dmg" download>PashtoDAOPKeyboardInstaller.dmg</a>link to start downloading the file directly to your computer.

**File Size**: ~26KB (DMG installer package)

### Step 2: Run the Installer

1. Locate the downloaded `PashtoDAOPKeyboardInstaller.dmg` file in your Downloads folder
2. Double-click on the DMG file to mount it
3. Double-click on the "PashtoDAOPKeyboardInstaller" inside the mounted volume
4. If macOS shows a message that the installer can't be opened due to security reasons, please follow the following steps:

- Open System Preferences → Privacy & Security → Security on newer macOS).
- At the bottom you’ll see “PashtoDAOPKeyboardInstaller.dmg was blocked to proectec your mac.” Click "Open Anyway".
- Now try opening the app again — you’ll get a prompt with an Open button. Click Open.
  ![pashto dao keyboard permission](/static/images/projects/pashtodao_allow_keyboard.jpg)

5. Click "Continue" when prompted
6. Follow the installation wizard to complete the setup

### Step 3: Restart Your Computer

After installation, restart your Mac to ensure all system changes take effect properly.

### Step 4: Use the Keyboard

Click on the keyboard icon in the menu bar and select "Pashto Phonetic By PashtoDAO"

## Troubleshooting

### Keyboard Not Appearing in Input Sources

1. Make sure you restarted your computer after installation
2. Check if the keyboard appears under "Other" in the input sources list
3. Try reinstalling the keyboard package
